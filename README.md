# netadmin_digibox
G-MediaNetAdmin

G-MediaNetAdmin for telkom digibox

Managing DigiBOX as easy as 123

Features :
1. Setup Network Address
2. Setup Hotspot
3. Setup DHCP Server
4. Setup DNS Hosts
5. Setup VirtualBOX VMs ( or LXC )
6. Router / Gateway
7. Surf Save ( DNS server from Save Source )
8. Antivirus (optional) for POP / SMTP proxy
